Поток событий

Краткое описание:

1.	Вариант пользования: «Chachok autoparts» пользователь может пользоваться каталогом для подбора и покупки автотоваров.
2.	Вариант пользования: «Chachok autoparts» пользователь может пользоваться сервисом в качестве продавца.



Предусловия:

 Для выполнения одного из вариантов пользования пользователю необходимо выбрать соответствующий раздел при регистрации на сайте.

Основные потоки событий:

«Chachok autoparts» пользуется покупатель:
1.	До регистрации пользователь имеет доступ к каталогу и инструменту сравнения товаров;
2.	Пользователь при регистрации выбирает пункт «Я покупатель»;
3.	Пользователь вводит свои данные;
4.	Пользователь нажимает на кнопку «Зарегистрироваться»;
5.	Сайт проверяет корректность введенных пользователем данных и если есть ошибка, то выводит её на экран;
6.	После регистрации пользователь получает доступ к корзине и получает возможность оформить заказ;
7.	После оформления заказа пользователь может отслеживать его статус на сайте;

«Chachok autoparts» пользуется водитель такси:
1.	Пользователь при регистрации выбирает пункт «Я продавец»;
2.	Пользователь вводит свои данные, включая номер личного соглашения с администрацией онлайн-каталога;
3.	Пользователь нажимает на кнопку «Зарегистрироваться»;
4.	Приложение проверяет корректность введенных пользователем данных, наличие подписанного соглашения и если есть ошибка, то выводит её на экран;
5.	Пользователь имеет право добавлять товары в каталог, а также добавлять свои предложения для уже существующих товаров.
6.	Во вкладке заказы продавец видит входящие заказы, которые может принимать и отклонять;
7.	После принятия заказа пользователь изменяет статус заказа по мере его выполнения;
8.	Пользователь видит оценки своего магазина и может отслеживать рейтинг товаров;

Альтернативные потоки событий:

Оценка заказа и товара:
1.	После выдачи заказа пользователь может оценить товар и оставить отзыв;
2.	После выдачи заказа пользователь может оценить продавца и оставить отзыв.

История заказов:
1.	Пользователи-покупатели имеют возможность просмотреть историю заказов, данные водителя.
2.	Пользователи-продавцы имеют возможность просмотреть историю продаж, выручку и данные покупателя.

Постусловия:
Постусловий для совершения заказов нет.
